Practica #10-Worker Pool en Go

Este programa implementa un sistema de (worker pool) en Go, utilizando goroutines, canales, contexto y WaitGroup.

Características

Procesamiento concurrente de tareas  
Uso de sync.WaitGroup 
Uso de context.Context para cancelación  
Medición del tiempo total de ejecución  

Usar el siguiente comando para ejecutar
go run main.go
